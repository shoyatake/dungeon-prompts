## 制約・前提

### ブランドガイドライン（絶対遵守）

- 配色: ダーク (#0a0f1e) + ゴールド (#f5c842) + アクセント紫 (#7c3aed)
- フォント: Cinzel (セリフ・装飾) / Noto Sans JP (本文) / Poppins (数字・英語)
- 視覚シンボル: SVG幾何シンボルのみ（絵文字・RPGアイコン禁止）
- 敵のネーミング: 言語現象の擬人化（Chain Slime等のRPG固有名詞禁止）
- sho先生: 日本語ガイダンスのみ（英語を話させない）

### デザイン原則

1. モバイルファースト (375x812 の iPhone サイズを基準)
2. 親指リーチ: 主要アクションは画面下1/3に配置
3. 片手操作: 通勤電車で使える前提
4. 認知負荷最小: 1画面1アクション
5. ゲーム感 x 品格: 中高生にダサく見えず、保護者に軽薄に見えない
6. アクセシビリティ: コントラスト比 4.5:1 以上、タップターゲット 44x44pt 以上

### 絶対にやらないこと

- 絵文字アイコン
- キャラクターイラスト
- アニメ調イラスト
- 「勇者・魔王・ドラゴン」等のファンタジー語
- 明るい背景
- 丸ゴシック多用

## 設計する9スクリーン

1. Portal - ダッシュボード・全体俯瞰
2. World Map - 10ダンジョン選択
3. Dungeon Detail - 1ダンジョン内ステージ一覧
4. Stage - 個別ステージ (既存 stage_v2.html)
5. Proving Ground - 定点観測・特別UI
6. Mission Hub - ミッション一覧
7. Mission Player - 7日間進行UI
8. Bookshelf - 語彙 + 修了証 + 観測履歴
9. Victory - クリア時の共通演出画面

## 9スクリーン詳細要件

### Portal
- 現在のXP・連続ログイン日数
- メインクエスト: 次のステージへのショートカット
- 定点観測カウントダウン or 「今日観測日です」バナー
- 進行中ミッション + おすすめ1つ
- 本棚への導線（獲得語彙数）
- 肝: 3本柱を対等に見せ、今日のTODOが1つに絞られる

### World Map
- D1〜D10を縦スクロールで俯瞰
- 状態: locked/current/cleared
- ジグザグ配置でゲーム感
- SVGで各ダンジョンのモチーフ表現

### Dungeon Detail
- ダンジョン名・説明・音ルールサマリー
- Tier 1〜5のセクション分け
- 各ステージ: 番号・タイトル・状態・星数
- ボスステージは最下部に大きく

### Stage
- P01の対象外（既存 stage_v2.html を参照）

### Proving Ground
- 重厚な入口演出
- 18文全体のプレビュー
- Before/After/初回の3点比較
- 成長スコア表示
- 他ステージと違う特別感（余白多め・フォント大きめ）

### Mission Hub
- Featured: おすすめ1つ（大きく）
- In Progress: 進行中（カレンダー風）
- Free: 音ルール別10本カード
- Premium: 価格付きカード
- Free/Premiumが対等に見える

### Mission Player
- カレンダー型7日進行
- Day 1診断録音、Day 4中間チェック、Day 7卒業録音
- 前日に戻れる、翌日はジャンプ不可
- 各日の所要時間表示

### Bookshelf
- 語彙カード（最大400語+ミッション分）
- カテゴリフィルター（品詞/テーマ/取得日）
- 修了証・成長の証（録音履歴）
- 統計（総XP・連続日数・クリアダンジョン数）

### Victory
- 大きな達成演出（SVGアニメーション）
- 獲得XP・語彙カード・称号
- Before/After並列再生
- 次のアクションへの導線
- SNS共有ボタン（任意）
